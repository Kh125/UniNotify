# UniNotify – Context-Aware Student Information System

UniNotify is a context-aware mobile/web application designed to help university students access important information such as class schedules, weather updates, announcements, and alerts in a centralized and user-friendly platform.

---

## Overview

Students often struggle to manage scattered information across different platforms (notice boards, emails, schedules). UniNotify solves this problem by providing a **unified system** that delivers real-time, relevant information based on user context.

---

## Objectives

- Provide centralized access to university-related information
- Improve student awareness of schedules and important updates
- Deliver real-time notifications for critical events
- Enhance user convenience through a context-aware system

---

## Problem Statement

Students face several challenges:

- Difficulty accessing real-time class schedules
- Missing important announcements and tasks
- Information scattered across multiple platforms
- Lack of real-time weather and situational awareness
- Security and privacy concerns for student data

---

## Solution

UniNotify provides:

- **Integrated Schedule Management** – easy access to class timetables  
- **Real-time Notifications** – alerts for tasks, announcements, and updates  
- **Weather Integration** – real-time environmental awareness  
- **Context-Aware Features** – information based on user location and time  
- **User-Friendly Interface** – simple and accessible design
